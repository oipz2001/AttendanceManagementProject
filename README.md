# How to run project.
Run mobile app (Android OS).
1. cd CMU_Attendance_Mobile_App
2. yarn install
3. yarn run android

Run Web app.
1. cd CMU_Attendance_Web_App
2. yarn install
3. yarn start

Run Server.
1. cd CMU_Attendance_Server\function-firecast\functions
2. yarn install
3. firebase emulators:start
